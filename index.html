Google Calendar sync

Connect your Google Calendar to see all your real events directly in this app. Since your Apple Calendar syncs to Google Calendar, all your Apple events will appear here too.

Your calendar data stays between you and Google. Nothing is stored on any server.

iCloud Calendar sync

Sync all calendars with Apple Calendar across iPhone, iPad, Mac, and Apple Watch.

Credentials stored only in your browser. Never sent anywhere except Apple's servers.
Generate at appleid.apple.com under Sign-In & Security

Setup steps

  1. Go to appleid.apple.com and sign in.
  2. Under Sign-In & Security, click App-Specific Passwords.
  3. Click + and label it "Life Organizer". Copy the password.
  4. Paste it above and click Save & connect.

AI Assistant API Key

Enter your Anthropic API key to enable the AI assistant. It is stored only in your browser and never shared with anyone.

Your key is saved locally on this device only. Never share it with anyone.

How to get an API key

  1. Go to console.anthropic.com and sign in.
  2. Click API Keys in the left sidebar.
  3. Click + Create Key and name it "MyLifeOrganizer".
  4. Copy the key and paste it above. Click Save.

Firebase setup

Enables real-time collaboration across all workspaces with Google sign-in.

Without Firebase the app works fully for one person. Firebase is only needed for multi-user real-time sync.
  1. Go to console.firebase.google.com and add a project named "MyLifeOrganizer".
  2. Enable Authentication and turn on the Google sign-in provider.
  3. Create a Realtime Database in test mode.
  4. In Project Settings, add a Web app and copy the config object shown.
  5. Paste the config below and click Save.
/mylifeorganizer
  /companies/{uc|ut|pa|family|ro}
    /events /tasks /people
  /users/{uid}
  /activity/{id}
